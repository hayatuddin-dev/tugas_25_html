Mengenal Semantic Elements Di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blogm maka kita bisa menambahkan aside dibagian bawah postingan untuk menjelaskan identitas penulis artikel. Karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
